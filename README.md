# Personal Finance Snapshot

A clean, delightful personal finance tracker built with **React + TypeScript + Tailwind CSS**.

## Features

- Monthly income & expense overview
- Category-based expense pie chart & income vs expense bar chart (Recharts)
- Budget setting with animated progress bars & over-budget warnings
- Add/delete transactions (persisted in localStorage)
- Smooth modals with Framer Motion animations
- Toast notifications (react-hot-toast)
- Month navigation
- Responsive layout

## Tech Stack

- React 18 + TypeScript
- Tailwind CSS
- Recharts
- date-fns
- uuid
- framer-motion
- react-hot-toast

## How to Run

```bash
# Install dependencies
npm install

# Start dev server
npm start

# Build for production
npm run build
```
